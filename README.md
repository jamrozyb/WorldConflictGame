# WorldConflictGame
Postanowiłem  zaimplementować grę „Światowy Konflikt”
główne motywacje:
- ćwiczenie i głębsze poznanie testów jednostkowych JUnit5
- lepsze zrozumienie zasad projektowych SOLIDM w szczególności „Dependency inversion principle” w tym przypadku zastosowanie tej zasady pozwala na oddzielenie logiki gry, od interfejsu użytkownika

W przyszłości:

- Stworzenie webowego interfejsu użytkownika (Spring Boot) 

Użytkowe:<br>

Prywatnie wraz ze znajomymi bardzo lubię te grę(posiadamy oryginalne wersje), jednakże w powodu mieszkania w różnych miastach rozgrywka nie jest możliwa.  Stąd też pomysł zaimplementowania gry, docelowo w wersji online. 

Instrukcja i opis gry:
